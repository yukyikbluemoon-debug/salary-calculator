# 🛡️ ระบบคำนวณค่าแรง รปภ.

ระบบคำนวณค่าแรงสำหรับพนักงานรักษาความปลอดภัย (รปภ.) แบบอัตโนมัติ

## ✨ ฟีเจอร์

- 📅 รองรับ 3 รอบเงินเดือน:
  - งวดที่ 1: 24 เดือนก่อน - 7 เดือนนี้ (ครึ่งเดือนแรก)
  - งวดที่ 2: 8-23 เดือนนี้ (ครึ่งเดือนหลัง)
  - งวดเต็มเดือน: 24 เดือนก่อน - 23 เดือนนี้
- 🎯 ตั้งค่าวันที่อัตโนมัติตามวันปัจจุบัน
- 💰 คำนวณค่าแรงอัตโนมัติ:
  - ทำงานปกติ (12 ชม.) = 650 บาท
  - ลาป่วย (8 ชม.) = 400 บาท
  - วันนักขัตฤกษ์ (12 ชม.) = 1,300 บาท
  - ลา/หยุด = 0 บาท
- 🎁 โบนัสรายวัน 32 บาท/วันทำงาน
- 🔥 ทำงาน 7 วันติด ได้ค่าแรงเพิ่ม 1 วัน
- 👔 ค่าตำแหน่ง 600 บาท/เดือน
- 🏥 หักประกันสังคม 5% + ประกันชีวิต 34 บาท
- 📊 สรุปโบนัสรายไตรมาส

## 🚀 วิธีใช้งาน

### แบบที่ 1: เปิดไฟล์โดยตรง
1. ดาวน์โหลดไฟล์ `index.html`
2. เปิดด้วยเว็บเบราว์เซอร์ (Chrome, Firefox, Safari)
3. ใช้งานได้ทันที!

### แบบที่ 2: ใช้งานบน GitHub Pages
1. เข้าลิงก์: `https://yourusername.github.io/salary-calculator/`
2. ใช้งานได้ทันทีผ่านอินเทอร์เน็ต
